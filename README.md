# DraftTrax

A tiny local app for tracking an NHL playoff pool draft. Pure HTML + vanilla JS
— no build step, no server, no dependencies at runtime.

## Run it

1. Clone/download this folder.
2. Open `index.html` in a browser.

If your browser blocks the `fetch('./data/players.json')` call from a `file://`
URL (Firefox does by default; Chrome usually doesn't), start a throwaway server
in this folder instead:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Usage

- **Left-click** a player row to cycle `available → taken → mine → available`.
- **Right-click** or **Shift+click** to cycle backward.
- **Undo** button (or **Cmd/Ctrl+Z**) reverts the last change.
- **Reset** clears all picks after a confirm.
- State auto-saves to `localStorage` — reload the page and your picks persist.

## Refreshing player data

Data lives in `data/players.json` and is generated by a small Node script that
hits the public NHL API (`api-web.nhle.com`). Regenerate it anytime (e.g., if
you want updated regular-season stats) with:

```sh
node scripts/fetch-data.js          # defaults to 2026 playoffs
node scripts/fetch-data.js 2027     # another year
```

Requires Node 18+ (for global `fetch`).

## Files

```
index.html         - page shell
app.js             - all logic (render, state, localStorage)
styles.css         - layout
data/players.json  - generated; committed so the app works offline
scripts/fetch-data.js - regenerates data/players.json from the NHL API
```
